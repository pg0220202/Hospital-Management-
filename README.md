# Hospital Management System

## Introduction

The Hospital Management System is a web application built with Django that allows both patients and doctors to register and login. After successful login:
- **Patients** can view blogs uploaded by doctors and book appointments.
- **Doctors** can upload blogs and view appointments.

## Features

- **User Authentication**: Registration and login for both patients and doctors.
- **Patient Dashboard**: 
  - View blogs posted by doctors.
  - Book appointments with doctors.
- **Doctor Dashboard**: 
  - Upload and manage blogs.
  - View appointments booked by patients.

## Technologies Used

- **Backend**: Django
- **Frontend**: HTML, CSS, JavaScript
- **Database**: SQLite (default for Django projects, can be configured to use other databases)
- **Others**: Django Admin for administrative tasks
